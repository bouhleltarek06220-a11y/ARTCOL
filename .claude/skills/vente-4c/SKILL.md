# Vente 4C — entretien commercial B2B

Colonne vertébrale d'un cycle de vente B2B. **Les 4C = Contact → Connaître → Convaincre → Conclure.** Cette skill orchestre les méthodes spécialisées selon la phase.

## Quand l'activer
Préparer/mener un RDV, un appel de découverte, un closing ; structurer un pitch ou une proposition ; traiter une objection ; qualifier ou relancer un deal. Contexte PlayAzur / Lost Chapter : vendre à des **créateurs, lieux, sponsors**.

## Les 4 phases

### 1) CONTACT — ouvrir, créer le lien
- Objectif : confiance + cadre clair.
- Accroche en 2 phrases (qui / pour qui / bénéfice concret), puis **cadrage** : agenda, durée, « ça vous va si on… ? ».
- Adapter au profil **DISC** : Rouge = direct/résultats · Jaune = relation/vision · Vert = réassurance/pas-à-pas · Bleu = preuves/détails.
- ❌ Ne pas pitcher l'offre tout de suite.

### 2) CONNAÎTRE — découvrir & qualifier (la phase qui fait gagner les deals)
- Découverte **SPIN** : Situation → Problème → Implication → Need-payoff (faire émerger le **coût de l'inaction**).
- Qualification : **BEBEDC** (Besoin, Enjeu, Budget, Échéance, Décideurs, Concurrents) pour PME/ETI ; **MEDDIC** pour comptes stratégiques / cycle long.
- Repérer les leviers **SONCAS(E)** = le *pourquoi* il achète (Sécurité, Orgueil, Nouveauté, Confort, Argent, Sympathie, Environnement).
- Reformuler systématiquement (« si je comprends bien… »).
- **Sortie attendue** : fiche de qualif remplie + 1 douleur prioritaire **chiffrée** + critères de décision + processus de décision.

### 3) CONVAINCRE — argumenter & lever les objections
- Argumenter en **CAP/CAB** : Caractéristique → Avantage → Preuve (rationnel) **ou** Bénéfice (émotionnel). Brancher chaque bénéfice sur le **levier SONCAS dominant** (« CAP SONCAS »).
- Preuves : cas clients, chiffres, démo, ROI, garanties.
- Objections en **CRAC** : Creuser → Reformuler → Argumenter → Contrôler. Ne jamais contrer de front ; isoler la **vraie** objection (prix ? confiance ? timing ? pouvoir de décision ?).
- Décideur débordé → cadrer l'offre **SNAP** (Simple, iNvaluable, Aligned, Priority).

### 4) CONCLURE — engager, négocier, closer
- Détecter les **signaux d'achat** → proposer la conclusion (choix alternatif, prochaine étape datée).
- Négociation **BATNA/MESORE** : connaître ta MEilleure SOlution de REchange et celle du client, viser la **ZOPA**, défendre la **valeur avant le prix**, ne concéder qu'en **échange**.
- Verrouiller : next step concret (qui / quoi / quand) + compte-rendu écrit.
- ❌ Ne jamais lâcher du prix sans contrepartie, ni terminer sans next step daté.

## Check-list de RDV (à produire)
- [ ] Accroche + cadrage
- [ ] 5–7 questions SPIN ciblées
- [ ] Grille de qualif (BEBEDC ou MEDDIC)
- [ ] 3 arguments CAB alignés SONCAS + preuves
- [ ] 3 objections probables + réponses CRAC
- [ ] Plan de négo (MESORE, plancher, concessions/contreparties)
- [ ] Proposition de next step datée

## Livrables typiques
Plan d'entretien 4C, trame d'appel, script de découverte, battle-card, fiche argumentaire, réponses d'objection, préparation de négociation.
