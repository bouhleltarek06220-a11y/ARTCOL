SENSOR — démo

Cette page charge le widget de chat embarquable. Renseignez l'URL de votre API et l'identifiant du client, puis cliquez sur « Charger le widget ». La bulle de chat apparaît en bas à droite.

1. Paramètres du widget

Astuce : démarrez l'API avec sensor serve et créez le client demo avec sensor init-client --id demo --nom "Démo", puis ingérez un document.

2. Intégration sur un vrai site

Une seule balise <script> à coller avant </body> : 

<script
  src="https://votre-api/sensor-widget.js"
  data-client="demo"
  data-api="http://localhost:8000">
</script>

Transparence AI Act : le widget indique en permanence qu'il s'agit d'un assistant IA.
