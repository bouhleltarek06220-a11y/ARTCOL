# Œuvres de la galerie

Déposez ici les images des tableaux, nommées **exactement** :

- `art-1.jpg` — Braise
- `art-2.jpg` — Horizon Captif (format large)
- `art-3.jpg` — Sève
- `art-4.jpg` — Nocturne Pourpre (format large)
- `art-5.jpg` — Or Liquide

Formats acceptés : `.jpg` (le code pointe sur `.jpg` ; pour `.png`/`.webp`,
adaptez le champ `image` dans `components/villa/world/artworks.ts`).

Conseils :
- Ratio conseillé ~3:4 (portrait) ou 3:2 (les deux « wide » : art-2 et art-4).
- Taille raisonnable (≤ ~1600 px de large) pour garder le chargement fluide.

Si un fichier est absent, le cadre affiche simplement un aplat de couleur — rien
ne casse.
